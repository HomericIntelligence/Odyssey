# Shared Directory

This directory contains reusable components and utilities that can be used across multiple paper implementations.

## Structure

- **core/** - Fundamental building blocks (tensor operations, basic algorithms, data structures)
- **training/** - Training utilities (optimizers, schedulers, training loops)
- **data/** - Data processing code (loaders, preprocessors, augmentation)
- **utils/** - General helper functions (logging, metrics, visualization)

## Usage

Import shared components in your paper implementations to avoid duplication and maintain consistency across the codebase.
