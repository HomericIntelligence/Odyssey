## What Worked

- **AST-based discovery** (bulk-test-type-annotation, testing-ast-import-layer-enforcement): Use `ast.parse()` with `ast.walk()` to find symbols at any nesting depth; test in subprocess for isolation
- **Grep-based verification**: After migration, use `grep -rn "old_module\."` to verify zero remaining references (from dry-consolidate)
- **Comprehensive reference updates**: Update imports, tests, CI configs, documentation simultaneously — don't fix piecemeal
- **Idempotent scripts**: Scripts that check "already done" before applying transformations (from bulk-test-type-annotation, src-layout migration)
- **Pre-commit hooks as safety net**: After bulk changes, run `pre-commit run --all-files` to catch formatting/linting issues that reveal incomplete migrations

## What Failed

- **Testing import-only changes with `importlib`** (testing-ast-import-layer-enforcement): `sys.modules` caching causes re-imports to silently succeed even if the module is partially broken; use `subprocess.run([sys.executable, "-c", "..."])` with clean state
- **AST walk only at top-level** (testing-ast-import-layer-enforcement): Missed function-local imports; always use `ast.walk(tree)` (recursive) not `tree.body` (top-level only)
- **Renaming at the producer instead of consumer** (architecture-mojo-import-ambiguity-alias-pattern): Breaks public API stability; fix collisions at import sites with aliases instead
- **Piecemeal updates**: src-layout migration showed that forgetting one `--cov=` path in README causes CI consistency checks to fail; comprehensive grep sweep catches these
- **Splitting tests to avoid collisions**: Shrinking test scope defeats regression detection; instead alias at import site

## Recommended Parameters

- **Grep patterns for bulk discovery**:
  ```bash
  grep -rn "from shared.tensor.any_tensor import" . --include="*.py" --include="*.mojo"
  grep -rn "from projectodyssey.shared.tensor.any_tensor import" . --include="*.py"
  ```
- **Verification sweep after migration**:
  ```bash
  grep -rn "from shared.tensor.any_tensor import\|from projectodyssey.shared.tensor.any_tensor import" . --include="*.py" --include="*.mojo"
  # Should return zero matches if migration is complete
  ```
- **AST-based validation** (similar to testing-ast-import-layer-enforcement pattern):
  ```python
  import ast, pathlib
  for py in pathlib.Path("src/").rglob("*.py"):
      tree = ast.parse(py.read_text())
      for node in ast.walk(tree):
          if isinstance(node, ast.ImportFrom) and (node.module or "").startswith("shared.tensor.any_tensor"):
              print(f"{py}:{node.lineno}  from {node.module} import ...")
  ```
- **Pre-commit full sweep**: Before each commit, run `pre-commit run --from-ref origin/main --to-ref HEAD` (not partial runs) to catch all affected files
- **Subprocess test for import verification** (from testing-ast-import-layer-enforcement):
  ```python
  subprocess.run([sys.executable, "-c", "from shared.tensor.tensor_creation import zeros, ones; print('OK')"], 
                 capture_output=True, check=True)
  ```
